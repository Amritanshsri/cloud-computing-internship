# Task 1 - Deploy a Static Website on AWS S3

## Objective

Host a static website using AWS S3 and make it publicly accessible.

## Services Used

* Amazon S3
* Static Website Hosting

## Steps Performed

1. Created an S3 bucket.
2. Uploaded index.html file.
3. Enabled Static Website Hosting.
4. Configured bucket policy for public access.
5. Accessed website through the generated endpoint.
